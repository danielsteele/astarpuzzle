A* Algorithm Puzzle Solver

  • The purpose of this project was to implement the A* algorithm, apply it to the puzzle, find a solution and output some interesting data.
  • I first implemented this algorithm in C++, in the Artificial Intelligence paper at Massey University in 2016 Semester 1.
  • This program is written entirely in JavaScript, CSS and HTML.
  • To avoid unsolvable postions leading to a memory exhaustion the shuffle function is fudged to only ever generate solvable boards.
  • The book "Eloquent JavaScript" was an excellent resource that I referred to while making this project.
  • The Board prototye and A* algorithm impementation are discussed in the Code section.

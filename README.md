# astarpuzzle
The A* search algorithm is used to solve a sliding tile puzzle.

Program Author:  
    Daniel Steele

Program Description:  
    The user is presented with a sliding tile puzzle.
    The user can move the tiles on the puzzle.
    The A* algorithm is used to solve the puzzle.
